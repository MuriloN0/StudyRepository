Atenção: Algumas linhas estão sobre efeito do assunto que estão abordando!

  1. Falando sobre tags

    Voltar
  • A primeira tag que aprendemos é a “i” que faz com que o texto fique em fonte itálico.
  • Em segundo aprendemos sobre a tag “Strong” que faz com que o texto fique em negrito.
  • Depois a tag “input” para colocarmos uma caixa de texto.
  • Vimos em seguia a tag “img” para atribuirmos uma imagem a página.

  1. Atributos das tags
    Voltar

  • Nós vimos um “Id” que é um identificador de elemento.
  • Vimos o a tag “style” que é para a formatação do seu html. Ele serve também para colocar um comando direto no seu elemento quando aplicado em uma tag direta no html.
  • Aprendemos o “Class” que também é um identificador que podemos usar em mais de uma tag.

  1. Introdução a Textos

    Voltar
  • “H1” que é a tag que usamos para o título da página.
  • Vimos que o “h1” tem uma variação até “h6” que controla o tamanho dos textos, fazendo assim um subtítulo.

  • Nos temos a tag “p” que se refere a um paragrafo.

  • Temos a tag “blockquote” para aquelas frases de efeito (uma citação ou uma explicação) que queremos colocar no texto.
  • Vimos a tag “u” para deixar um underline na palavra, ou frase.
  • a tag “mark”, que vai fazer um sublinhado como marca texto.
  • Podemos usar a tag “sup” para colocarmos uma letra ou palavra flutuando na parte de cima do campo da linha.

  1. Listas Ordenadas e não Ordenadas

    Voltar
  • Nós temos na lista ordenada a tag “ol” que será o contêiner para manter ela ordenada. Dentro dela será colocada a tag “li”, que vai ficar os itens, como parágrafos e afins.
  • Na lista não ordenada colocamos a “ul”, que é o contêiner para os itens dentro da lista. Dentro dela fica a tag para os itens que será o “li”.

  1. Entendendo os Links

    Voltar
  • A tag “a” é uma ancora, onde queremos dizer que ele está preso a uma página específica, e fica ancorado a ela.
    • Temos dentro dele o “href”, que é onde colocamos o link dele.
    • Podemos usar também o “target” que é usado para falar se ele vai abrir na própria guia, ou se abre em uma nova.
    • Outro que temos é o tile que mostra aquele balão de mensagem quando paramos o mouse em cima de um link e afins.
